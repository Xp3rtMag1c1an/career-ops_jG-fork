## ⚡ Quick Setup (One-Time)

```bash
# Install dependencies (one time)
npm install marked playwright
npx playwright install chromium

# Generate ALL PDFs in one go
for dir in applications/*/; do
  node md-to-pdf.mjs "${dir}cv.md"
  node md-to-pdf.mjs "${dir}cover-letter.md"
done
```

## 🧭 Workflow per Application

1. `cd` into the folder (e.g. `cd applications/00-zapier-automation-strategist/`)
2. Open `apply.md` — has the URL, salary expectations, and pre-written form answers
3. Open the URL in your browser
4. Upload `cv.pdf` (already generated by the for-loop above)
5. Either upload `cover-letter.pdf` OR copy-paste the markdown into the form
6. Copy-paste form answers from `apply.md` into the matching fields
7. Submit
8. Move to next folder

**~3-5 minutes per application once PDFs are generated.**
